# 🌾 Smart Agriculture Bot

An AI-powered agriculture assistant built using Python, Streamlit, and Google Gemini AI.

## 🚀 Features

* Crop recommendations
* Farming guidance and tips
* Agriculture-related question answering
* Interactive chatbot interface
* Chat history management
* User-friendly Streamlit web application

## 🛠️ Technologies Used

* Python
* Streamlit
* Google Gemini API
* Git & GitHub

## 📂 Project Structure

```text
Agriculture_Bot/

├── app.py
├── chatbot.py
├── prompt.py
├── requirements.txt
├── .env.example
├── README.md
└── chat_history.json
```

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/geethapriya412/Agriculture_Bot.git
```

2. Move to the project directory

```bash
cd Agriculture_Bot
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Create a `.env` file and add your Gemini API key

```env
GOOGLE_API_KEY=YOUR_API_KEY
```

5. Run the application

```bash
streamlit run app.py
```
